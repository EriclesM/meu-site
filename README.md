🌐 Portfólio Pessoal – Éricles Marçal

Este projeto é um site de portfólio pessoal desenvolvido para apresentar minhas informações, formação acadêmica e projetos na área de tecnologia.

🚀 Sobre o projeto

O site foi criado utilizando HTML, CSS e JavaScript, com foco em organização, design moderno e boa experiência do usuário. Possui múltiplas páginas e navegação simples e intuitiva.

🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript

✨ Funcionalidades

Navegação entre páginas (Sobre mim, Formação, Portfólio e Contato)

Layout responsivo com Flexbox

Tema claro e escuro (Dark Mode)

Armazenamento de preferência de tema com localStorage

Formulário de contato com validação em JavaScript
